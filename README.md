# Traductor Broteñol

Este proyecto muestra un ejemplo muy sencillo de traducción del español a un idioma ficticio llamado **broteñol** basado únicamente en diccionarios y reglas locales. No se utilizan APIs ni modelos de IA externos, por lo que funciona completamente gratis en GitHub Pages.

## Uso
1. Abre `index.html` en tu navegador (no se necesita conexión a Internet).
2. Escribe un texto en español y pulsa **Traducir**.
3. Verás la traducción generada siguiendo las reglas definidas en `script.js`.
